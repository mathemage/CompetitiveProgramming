Problem Statement

    

You are planning some airline travel. Direct flights are offered between several cities, but not necessarily between every pair of cities. Each flight that is offered takes a certain travel distance. Your goal is to find the shortest total distance possible, using any number of connecting flights, to get from start city to end city.

You are also given the distances for each available flight, in vector <string> dist. For each element i, the character j indicates the distance travelled from city i to city j. A distance of 0 means there is no flight available in that direction between those two cities. Note that the distance may not be the same in both directions.

Return the minimum distance travelled from city start to city end. If it is not possible to make such a trip, return -1.

Definition

    
Class:AirlineTravelPlanner
Method:shortestDistance
Parameters:int, int, vector <string>
Returns:int
Method signature:int shortestDistance(int start, int end, vector <string> dist)
(be sure your method is public)

Limits

    
Time limit (s):2.000
Memory limit (MB):256

Constraints

-start and end will be between 0 and Size(dist) - 1, inclusive.
-dist will contain between 2 and 20 elements, inclusive.
-Each element of dist will be the same length as the number of elements in dist.
-Each character in each element of dist will be '0'-'9', inclusive.
-Each character i of element i of dist will be '0'.
